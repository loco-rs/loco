Loco 是一个用 Rust 编写的 Web 框架，类似于 Rails。Loco 提供快速构建 Web 应用的功能，并且允许创建自定义任务，可以通过 CLI 运行。

## 特性

- **简单的 API**: 使用 Rust 的强类型系统确保安全性和可靠性。
- **快速开发**: 提供快速构建 Web 应用的工具和模板。
- **CLI 支持**: 可以创建和运行自定义 CLI 任务。
- **灵活性**: 支持自定义配置和扩展。

## 安装

通过 Cargo 安装 Loco:

```sh
cargo install loco
```

## 快速开始

创建一个新的 Loco 项目:

```sh
loco new my_project
cd my_project
```

启动开发服务器:

```sh
cargo loco start
```
